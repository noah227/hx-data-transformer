# 一键变换

一个小工具

## 功能

- [x] 变对象
- [x] 变数组
- [x] 变风格
- [x] 变二维码（现在支持GUI的模式了）
- [x] 编码/解码（URI）
- [x] 注释翻转
- [x] 对象json修正（beta）
